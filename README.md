# Random Task Generator

Автор: Шибаева Ксения Сергеевна

## Описание
Это графическое приложение, которое генерирует случайные задачи из предопределенного списка. Пользователь может выбрать тип задачи, просматривать историю сгенерированных задач, а также добавлять новые задачи.

## Как пользоваться
- Нажмите кнопку «Сгенерировать задачу» для получения случайной задачи по выбранному фильтру.
- Используйте выпадающее меню для фильтрации задач по типу (Все, учёба, спорт, работа).
- Для добавления задачи в список введите текст, выберите тип и нажмите «Добавить задачу».
- Вся история сохраняется в файл `tasks.json` и загружается при запуске.

## Примеры
- Генерация случайной задачи
- Добавление новых задач
- Фильтрация результата

1. Склонируйте репозиторий
2. Установите Python (если ещё не установлен)
3. Запустите файл `main.py`
